  <action>
    Create scripts/lib/render-panel.js. CommonJS ('use strict', module.exports). No new npm dependencies — chalk@4.1.2 only.

    **Constants (from CONTEXT.md locked decisions):**
    ```javascript
    const BOX_WIDTH = 64;           // fixed panel width above narrow threshold
    const NARROW_NO_ART = 60;       // below: omit mosque art, keep box frame
    const NARROW_NO_BOX = 40;       // below: fall back to plain text (Claude's discretion)
    ```

    **Chalk instance (Pitfall 3 fix — force color in non-TTY hook context):**
    ```javascript
    const chalk = require('chalk');
    const c = new chalk.Instance({ level: process.env.NO_COLOR ? 0 : 3 });
    const green = s => c.hex('#2d6a4f')(s);   // Islamic green — readable on dark + light (RESEARCH.md Pitfall 7)
    const dim   = s => c.dim(s);
    ```

    **Box characters:**
    ```javascript
    const B = { tl:'\u250C', tr:'\u2510', bl:'\u2514', br:'\u2518', h:'\u2500', v:'\u2502' };
    ```

    **Mosque art (two variants):**
    - MOSQUE_ART_STANDARD: 5–7 lines, minimalist dome + minaret silhouette, fits 62 inner chars (BOX_WIDTH - 2 border chars). Hand-crafted; no figlet. Example structure:
      ```
          |      |      |
         _|_    _|_    _|_
        /   \  /   \  /   \
       |     ||     ||     |
      ~~~~~~~~~~~~~~~~~~~~~~~~~~~
      ```
      Design it so each line is <= 62 chars (inner width = BOX_WIDTH - 2). Apply `green()` to each art line.
    - MOSQUE_ART_SMALL: omit entirely (cols < NARROW_NO_ART) — the "small variant" per CONTEXT.md is: no art, box frame still present.

    **renderPanel(ayah, opts) function:**
    1. `const cols = (opts && opts.cols) || 80`
    2. If `cols < NARROW_NO_BOX` (discretionary threshold ~40): return plain text (Phase 1 dashes style) — no box, no art
    3. Determine `width = cols < NARROW_NO_ART ? cols : BOX_WIDTH`
    4. Build content lines array:
       - If `cols >= NARROW_NO_ART`: prepend MOSQUE_ART_STANDARD lines + one blank line
       - Then: Arabic line, transliteration line (dim), translation line, reference line (green)
    5. Each text line: pad/truncate to `width - 4` chars (1 border + 1 space each side), then wrap in `green(B.v) + ' ' + line + ' ' + green(B.v)`
    6. Build top/bottom borders: `green(B.tl + B.h.repeat(width - 2) + B.tr)` etc.
    7. Join all lines with '\n', return as string

    **Content order (CONTEXT.md locked):** mosque art → blank line → Arabic → transliteration → translation → reference

    **Reference line format:** `\u2014 ${ayah.surah_name} ${ayah.surah_number}:${ayah.ayah_number}` — apply `green()` to full line

    **DISP-06 (Arabic as-is):** Do not transform, transliterate, or pad Arabic text. Include it verbatim.

    **Zero-crash guarantee:** Wrap entire renderPanel body in try/catch — on any error return a minimal fallback string (empty systemMessage-compatible string).

    **Export:** `module.exports = { renderPanel };`

    Commit message after GREEN: `feat(02-01): implement render-panel.js (GREEN)`
